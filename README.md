## 🚀 Project 1: Legacy Infrastructure Migration & Digital Transformation
**Context:** Restaurant Management Platform Modernization  
**My Lenses:** Business Analyst / Functional Analyst / Tech Lead

### 📋 Project Overview
Led the consulting effort to transition an outdated, legacy restaurant infrastructure platform into a modern, scalable web and mobile application. This project required a deep dive into technical debt, process optimization, and system architecture mapping.

### 🛠️ Key Contributions
* **Business & Functional Analysis:** Conducted detailed As-Is vs. To-Be analysis. Translated high-level business operational needs into concrete functional specifications and system workflows.
* **Technical Leadership:** Evaluated the old infrastructure constraints, guided the data migration strategy, and ensured the new application design aligned with technical best practices without interrupting business operations.
* **PMO & Governance:** Maintained project timelines, managed technical risks, and bridged communication between restaurant stakeholders and the development team.

## 🛸 Project 2: E-Commerce Platform Modernization & Redesign
**Context:** Drone Retail E-Commerce Website  
**My Lenses:** Functional Analyst / PMO Analyst / Business Analyst

### 📋 Project Overview
Drove the digital transformation of a legacy drone retail website into a highly functional, secure, and user-centric e-commerce platform designed to optimize conversion rates and streamline backend order fulfillment.

### 🛠️ Key Contributions
* **Requirements & UX Engineering:** Facilitated stakeholder workshops to extract business requirements. Created user journeys, wireframes, and detailed functional specifications for the checkout funnel and product filtering.
* **PMO Analyst Tracking:** Managed the project scope, tracked key milestones, and established delivery metrics to ensure the website redesign stayed aligned with the strategic timeline and budget.
* **Functional Mapping:** Authored clear, technical user stories and acceptance criteria, ensuring developers had a foolproof blueprint for integration (payment gateways, inventory tracking).
